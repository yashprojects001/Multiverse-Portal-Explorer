# 🌀 Multiverse Portal Explorer

![Multiverse Portal](https://img.shields.io/badge/Project-Multiverse%20Portal%20Explorer-purple?style=for-the-badge\&logo=github)

> 🌌 A creative web experiment where glowing portals allow you to jump between different universes.

---

## 🚀 Overview

**Multiverse Portal Explorer** is an interactive webpage that simulates **portals to different universes**.
Each portal transforms the environment into a new dimension with its own visual theme.

Instead of navigating through traditional pages, users **travel between worlds**.

---

## ✨ Features

🌀 **Glowing Portal System**
Four animated portals represent different universes.

🎨 **Dynamic Universe Themes**

| Portal      | Universe                   |
| ----------- | -------------------------- |
| 🌌 Cosmic   | Deep space galaxy          |
| 🟣 Cyber    | Futuristic cyberpunk world |
| 🌿 Nature   | Calm green ecosystem       |
| 🔺 Abstract | Vibrant artistic dimension |

🖱 **Interactive Hover Effects**
Portals expand and glow when hovered.

⚡ **Smooth Transitions**
Background transitions smoothly between universes.

💡 **Minimal Yet Futuristic Design**
Clean layout with neon portal effects.

## 🎮 How It Works

1️⃣ Four portals are displayed on the page.
2️⃣ Each portal has a **unique color and glow effect**.
3️⃣ When a portal is clicked:

* A JavaScript function activates
* The environment theme changes
* The user enters a different universe.
